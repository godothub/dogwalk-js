# README

This directory is the unchanged Godot project directiory of the game DOGWALK.
This includes the source code, comprised of Gdscript, JSON, Python and other Godot related file formats.

Art assets and animations are primarily exported as the .glTF file format with additional image file formats for textures.
The original .blend files that were used to create and export these files are not included in this directory.

For more info about the licencing, read the LICENCE.md document in this directory.

More information, documentaiton and open source files are accessbile via studio.blender.org, as well as other Open Projects from the Blender Studio.
